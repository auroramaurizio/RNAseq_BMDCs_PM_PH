# Bulk RNAseq R code
R scripts for the bioinfo bulkRNAseq data analyses described in:

Microbiome modulation uncouples efficacy and toxicity induced by immune checkpoint blockade in mouse multiple myeloma, L.L. Cogrossi et al., Nature Communications, 2025, DOI: [10.1038/s41467-025-65312-y](https://doi.org/10.1038/s41467-025-65312-y).

[![DOI](https://zenodo.org/badge/1064157106.svg)](https://doi.org/10.5281/zenodo.17234615)

RNAseq counts data have been deposited in GEO and are available under accession number GSE306088.
